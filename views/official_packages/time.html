Time Package

A library to standardize time formats across an instance.

Configure via config/time.json with the following parameters: 

{
  "default_timezone": null,
  "timestamp_format": "m/d/Y g:i:s A",
  "time_format": "g:i A",
  "date_format": "m/d/Y",
  "empty_time": "N/A",
  "sql_date_format": "%Y/%m/%d",
  "sql_time_format": "%h:%i %p"
}

Missing parameters will be replaced with the default (shown above).

If the timezone is not set, it will be auto-set to UTC.

Timezone can be set at runtime with the Time::setTimezone(string $timezone).

You can read more about formatting in the PHP documentation.

Example Usage

use DynamicSuite\Time;

$date = '2020-07-08 14:30:00';

echo Time::date($date); // 07/08/2020
echo Time::time($date); // 2:30 PM
echo Time::timestamp($date); // 07/08/2020 at 2:30 PM

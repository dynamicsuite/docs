PHPMailer Package

This is a Dynamic Suite package implementing the PHPMailer class for sending emails from Dynamic Suite.

The PHPMailer project can be followed on GitHub.

Configure via config/phpmailer.json with the following parameters: 

{
  "smtp_debug": 0,
  "smtp_host": "mail.example.com",
  "smtp_port": 587,
  "smtp_username": "noreply@example.com",
  "smtp_password": "12345",
  "smtp_secure": "tls",
  "default_from_addr": "noreply@example.com",
  "default_from_name": "My Name"
}

Missing parameters will be replaced with the default (shown above).

Example Usage

use DynamicSuite\PHPMailer\PHPMailer;

try {
    $email = PHPMailer::create();
    $email->addAddress('email@example.com', 'Example Account');
    $email->Subject = 'Test Email';
    $email->Body = 'This is a test!';
    $email->send();
    echo 'Mail sent';
} catch (\PHPMailer\PHPMailer\Exception $exception) {
    echo $exception->getMessage();
}

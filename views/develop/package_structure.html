Package Structure

A package is defined by its structure file which is a JSON file in the root of the package directory. The name of the structure file must be the same as the package ID with the .json file extension.

Note, if a parameter is not provided, the default will be used.

Parameters

The name of the package. The package author. Semantic version of the package. A short description of the package. The license type of the package. List of paths to include in the autoloader. List of paths to scripts to execute on every DS request. List of paths to JS files to include on all client views. List of paths to CSS resources to include on all client views.

API Structure

A package may contain "apis" which are REST endpoints.

Note, if a parameter is not provided, the default will be used.

Parameters

The entry point of the API. This is the path to a PHP script which will be executed when the API is requested. An array of strings of required POST keys for the API. An array of strings of required permissions on the session for the API. If the API is public and does not require authentication. An array of string paths to include in the autoloader for the API. An array of string paths of PHP scripts to execute for the API. These scripts are executed before the entry.

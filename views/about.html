What is Dynamic Suite?

Dynamic Suite is a general-purpose web application framework. It's main focus is on private login-required pages with enhanced security, but it is able to template public web pages as well (Like these docs!). The framework includes the tools needed to build a package-based web application with utilities to manage client views, server libraries, REST API endpoints, and much more!

Dynamic Suite is available under the GNU General Purpose License v3. A copy of the license is available on the GitHub repository.

Issues and support are maintained through GitHub issues within the repository.

Built for Performance

Dynamic Suite was built to perform in high traffic production environments.
